4. **File Uploads** - Avatar/document uploads not functional
5. **Export Features** - Buttons present but not wired
6. **Real Pagination** - UI present but not functional
7. **Form Validation** - Minimal validation implemented
8. **Error Handling** - No error states or boundaries
9. **Loading States** - No loading indicators
10. **Database Operations** - No CRUD functionality
