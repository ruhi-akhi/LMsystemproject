Smart Inventory Documentation

Thank you for purchasing Smart Inventory — a modern Next.js 15 inventory management template with dashboards, QR ordering, analytics, and role-based access control.

Stack Next.js 15 · React 18 · TypeScript · Tailwind CSS v4 · DaisyUI · MongoDB · Socket.IO

Requirements

  • Node.js 18.x or 20.x
  • npm 9+ or yarn
  • MongoDB (local or MongoDB Atlas)
  • Optional: Stripe, Cloudinary, Firebase, Gmail SMTP, Gemini API

Installation

1. Install frontend dependencies

npm install

2. Configure environment

Copy .env.example to .env.local and fill in your values.

cp .env.example .env.local

3. Install socket server (optional, for real-time chat)

cd backend/server
npm install

4. Run locally

Terminal 1 — Socket server:

cd backend/server
npm run dev

Terminal 2 — Next.js app:

npm run dev

Open http://localhost:3000

5. Production build

npm run build
npm run start

Demo Data (One-Click Setup)

On the login page, click Load Demo Data & Login. This will:

  • Seed demo admin user: admin@inventory.com / admin123
  • Create sample categories and products
  • Log you into the dashboard automatically

API endpoints:

POST /api/demo-data   # Create demo data
GET  /api/demo-data   # Check demo data status

Deployment (Live Demo)

Vercel (recommended for Next.js)

  1. Push project to GitHub
  2. Import repository in Vercel
  3. Add environment variables from .env.example
  4. Set NEXT_PUBLIC_APP_URL to your Vercel domain
  5. Deploy and test demo login flow

Socket server

Deploy backend/server to Railway or Render and set NEXT_PUBLIC_SOCKET_URL to the live socket URL.

MongoDB Atlas

Create a free cluster, whitelist Vercel IPs (or allow from anywhere for demo), and use the connection string in MONGODB_URI.

Main Features

  • Inventory dashboard with stock alerts
  • Product & category management
  • Order management & restock queue
  • QR scan-to-checkout demo flow
  • Role-based access: staff, manager, admin
  • Dark/light theme, responsive layout
  • Stripe checkout integration
  • AI chat assistant (Gemini)
